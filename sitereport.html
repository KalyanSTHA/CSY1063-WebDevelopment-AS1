Site Report

My Web Development Journey

I first learned about web development when I was in high school. Although I had a little experience back then, it was mostly limited to the very basics of HTML and CSS. Before starting this module, I still felt like a beginner and hadn't built any real projects yet. Because of this, the idea of building a complete portfolio website seemed quite overwhelming. However, as I progressed through the lessons, I began to understand the fundamentals of how the web works.

The Learning Process

In the initial weeks, I learned how to use GitHub and the basic structure of HTML documents. We practiced using semantic elements like header, nav, section, and footer to organize content properly. For me, CSS was the most challenging part at first. It was difficult to understand the Box Model and how padding, margins, and borders worked together to move elements on the page. I often found myself moving one thing only to have another move by mistake.

Challenges Faced

Creating the demonstration video was quite difficult for me, I had a few retakes for some portion of the video. The process of creating hamburger icon that only appears in mobile was very difficult for me. On the technical side, understanding media queries took some time. It was a struggle to make the website look good on a mobile screen while keeping the desktop version organized. I also spent a lot of time choosing a color theme. At last, I choose a warm brownish colour palette which I quite liked.

Process of Debugging

Initially, debugging was very frustrating, but it became easier with time. I learned how to use browser developer tools to inspect elements and why certain styles weren't applying correctly. Most of my issues were simple mistakes like typos in class names or forgetting to close tags properly. Overtime, I developed the habit of checking my code carefuly before testing for the final result.

Design Decisions

For the design of this portfolio, I focused on balancing the aesthetics and readability:

I used warm tones [#faebd7, #8b5a2a, #5c4033, #3b2f2f]. I got this set of colours from a blog written in Piktochart website's "Cream White Color Combinations". I chose Georgia for body text because it is appealing and Arial for the navigation for clean look. I implemented a centered navigation bar for easy access to switch between pages. Also, I experimented with various kind of border styles to change visuals in each page.

Skills Developed

Through this project, I have developed a solid foundation in HTML structure, CSS styling, responsive design, and problem solving. I now understand how to create complex layouts using Flexbox, apply CSS transitions for simple animations, and ensure that my websites work on different screen sizes.

Reflection

This module gave me understanding of web development. As my first project, I am proud of what I have created and I am exicited to create a lot more. Different challenges in design decisions and debugging process helped me to learn and grow, I look fordward to adding future projects to this portfolio and continue to improve my skills in coming time.

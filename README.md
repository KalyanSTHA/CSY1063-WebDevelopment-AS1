# CSY1063-WebDevelopment-AS1
This project is a portfolio website of Kalyan Shrestha. 
 Pages:

- **index.html** - Home page with introduction, photo gallery, and education section.
- **project.html** - Projects showcase page.
- **contactinfo.html** - Contact information and form.
- **videodemo.html** - Video demonstration page.
- **sitereport.html** - Site report and essay.

Usage:
1. Clone or download the project files.
2. Extract all files 
3. Run index.html and navigate between pages using navigation bar. 

Languages Used: 
HTML and CSS
